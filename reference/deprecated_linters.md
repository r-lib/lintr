# Deprecated linters

Linters that are deprecated and provided for backwards compatibility
only. These linters will be excluded from
[`linters_with_tags()`](https://lintr.r-lib.org/reference/linters_with_tags.md)
by default.

## See also

[linters](https://lintr.r-lib.org/reference/linters.md) for a complete
list of linters available in lintr.

## Linters

There are not currently any linters tagged with 'deprecated'.
