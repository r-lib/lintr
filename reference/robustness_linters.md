# Robustness linters

Linters highlighting code robustness issues, such as possibly wrong edge
case behavior.

## See also

[linters](https://lintr.r-lib.org/reference/linters.md) for a complete
list of linters available in lintr.

## Linters

The following linters are tagged with 'robustness':

- [`absolute_path_linter`](https://lintr.r-lib.org/reference/absolute_path_linter.md)

- [`all_equal_linter`](https://lintr.r-lib.org/reference/all_equal_linter.md)

- [`backport_linter`](https://lintr.r-lib.org/reference/backport_linter.md)
